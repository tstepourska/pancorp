Centralized CVS Directory Templates

Analysis and Design (analysis_design)

Overview:  
The analysis & design artifact set captures and presents information related to the solution to the problems posed in the requirements set.

Usage:  
It includes artifacts such as the data model, design model, software architecture document, analysis model, architectural proof-of-concept, deployment model, and reference architecture.
