Project Management (project_management)

Overview:
The project management artifact set captures the artifacts associated with project and process planning and execution.

Usage:
The project management folder will include artifacts such as the: business case, issues list, iteration assessment, iteration plan, review record, risk list, software development plan, status assessment, measurement plan, problem resolution plan, product acceptance plan, project measurements, quality assurance plan, risk management plan, and work orders.
