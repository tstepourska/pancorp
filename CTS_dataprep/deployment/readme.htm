Deployment (deployment)

Overview:   The deployment folder contains server-side files that will be migrated for deployment onto eBCI server.

Usage:   The deployment folder will include artifacts such as the: application ear deployment unit and xml files, server- and client-side ejb.jar files, deployment plan, end-User support material, release notes, training materials, bill of materials, deployment unit, installation Artifacts, product, and product artwork.
