Configuration Change Management (configuration_change_mgmt)

Overview:
The configuration & change management artifact set captures and presents information related to the configuration and change management discipline.

Usage:
The configuration & change management folder will include artifacts such as the: change request project repository workspace, configuration audit findings, and configuration management plan.
