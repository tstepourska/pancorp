Requirements (requirements)

Overview:
The requirements artifact set captures and presents information used in defining the required capabilities of the system.

Usage:
The requirements folder will include artifacts such as the: glossary stakeholder requests supplementary specifications use case model user interface prototype vision boundary class requirements attributes requirements management plan software requirements specifications use case storyboard Rational RequisitePro files and use cases, and Rational Rose data modeler files.
