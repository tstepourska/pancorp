Overview:
The artifacts developed as products of the test and evaluation .

Usage:
The test folder will include artifacts such as the: test evaluation summary, test plan, test results, test ideas list, test case, test data, workload analysis model, test environment configuration, test log, test script, test suite, test automation architecture, test class, test component, and, test interface specification
